# todoapp-nodejs
## 目的
todoappの作成を通してNode.jsの学習をすすめる

## 必要機能
- アカウント作成
- ログイン
- タスク
  - 登録
  - 完了
  - 削除

## 使用技術
- Node.js
  - Express
- MongoDB
- Figma
